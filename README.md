# xulang
小老鼠语言：一门极简、免记忆规则的编程语言。

## 词法单元
常量：由大写字母、数字、下划线构成的、至少具有一个大写字母或数字的字符串。例如 `123`, `ABX_1`, `_T` 等等。

变量：由大写字母、数字、下划线构成的、至少含有一个小写字母的字符串。例如 `a`, `Name`, `_var1`。

## 程序示例
```
// 定义 IF 三目运算符（具有短路性）
(IF TRUE  a b) => a
(IF FALSE a b) => b

// 定义 MERGE: 合并任意两个序列
(MERGE (*a) (*b)) => (*a *b)

// 定义 HEAD: 取非空序列首个元素
(HEAD (a *b)) => a

// 定义 REV: 对任意序列进行翻转
(REV ()) => ()
(REV (a *b)) => (MERGE (REV (*b)) (a))

// 定义 TAIL: 求非空序列的尾部元素
(TAIL (*a)) => (HEAD (REV (*a)))

// 运行程序（计算结果为 E）
(IF FALSE X (TAIL (A B C D E)))

// 运行程序（计算结果为 1）
(IF TRUE 1 (TAIL (A B C D E)))
```
